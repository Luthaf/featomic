Overview
########

This developer documentation is divided into three parts:

1. :ref:`devdoc-get_started` explains how you can start developing code and documentation
2. The :ref:`devdoc-how_to`, take you through a series of steps on key problems for developing 
   rascaline.
3. In the :ref:`devdoc-explanations` section discusses key topics and concepts at a fairly high 
   level and provides useful explanations to expand your knowledge about the
   architecture of rascaline.
